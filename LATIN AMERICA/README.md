# Latin America folder

## Purpose
This folder contains startup data for Latin America, especially Brazil, Mexico, and YC-linked companies from the region.

## Key files
- `brazil_startups.csv` — Brazil startup records
- `mexico_startups.csv` — Mexico startup records
- `LatinAm_brazil_columbia.xlsx` — consolidated Latin America workbook
- `yc_latam_companies.csv` — YC Latin America company list used for enrichment and status labeling

## How this folder is used
These files support the regional coverage for Latin America and provide supplementary examples for the thesis’s broader geographic comparison.
