# Extra datasets folder

## Purpose
This folder contains auxiliary datasets that were reviewed during the project for comparison, enrichment testing, or baseline evaluation.

## Key files
- `companies.csv` — broad company list dataset, later rejected due to poor funding coverage and regional imbalance
- `startup_success_dataset.csv` — dataset considered but rejected as synthetic or unsuitable for the thesis labeling task
- `startup_valuation_dataset.csv` — dataset considered but rejected for the same reason

## How this folder is used
These files helped the project evaluate alternative data sources and justify why the final thesis relied on the curated main dataset and regional supplements instead.
