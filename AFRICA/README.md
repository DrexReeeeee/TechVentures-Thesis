# Africa folder

## Purpose
This folder contains African startup funding datasets, raw source snapshots, and the cleaning pipeline used to build the Africa supplement for the thesis.

## Key files
- `AFRICA_2022.xlsx` — earlier Africa dataset snapshot for 2022
- `Funded African tech startups 2022.xlsx` — raw 2022 funding data snapshot
- `Funded African Tech Startups 2023 (by country and by sector).xlsx` — raw 2023 funding data snapshot
- `FINAL AFRICA/` — cleaned final Africa supplement folder used in the project dataset
- `Funded_african_2024/` — 2024 materials folder
- `Funded_African_tech_startups_2025_(by_country-sector)/` — 2025 materials folder
- `merge_and_clean_africa_data.py` — Python script for cleaning and merging the Africa data

## How this folder is used
These files were used to build the Africa-specific supplement for the regional startup success dataset and to support the thesis discussion on non-Western startup ecosystems.
