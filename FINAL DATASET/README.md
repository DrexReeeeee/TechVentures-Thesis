# Final dataset folder

## Purpose
This folder contains the main modeling-ready datasets and the final merged master files used for training, evaluation, and regional analysis.

## Key files
- `big_startup_secsees_dataset.csv` — main base dataset that forms the backbone of the project
- `final_master_dataset.csv` — final merged master dataset used for analysis
- `final_master_dataset_updated.csv` — updated version of the final master dataset
- `india_startups_cleaned.csv` — cleaned India startup dataset
- `investments_VC.csv` — enrichment dataset with funding-round detail
- `regional_distribution_summary (1).xlsx` — regional distribution workbook
- `regional_distribution_summary_FINAL.xlsx` — final regional distribution workbook

## How this folder is used
This folder is the core of the thesis pipeline. It contains the labels, features, merged regional data, and summary files used to train and evaluate the TechVenture-SFM model.
