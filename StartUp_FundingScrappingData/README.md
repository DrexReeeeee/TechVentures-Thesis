# Startup funding scraping data folder

## Purpose
This folder stores the yearly startup funding data collected by the scraping pipeline and the scripts used to merge those files into consolidated outputs.

## Key contents
- `2015/` to `2021/` — yearly folders containing raw CSV files collected for each year
- `merged_output/` — merged outputs generated from the yearly scrape data
- `merge_csvs.py` — script used to combine the yearly CSVs into consolidated files

## How this folder is used
These files support the project’s funding-history dataset and were used as part of the broader data collection pipeline for the thesis.
